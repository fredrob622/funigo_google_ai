Dialogue à deux voix avec synthèse vocale


// Utilisation Deux menus déroulants te permettent de choisir deux voix différentes parmi celles installées sur ton système pour le dialogue. Modifie les répliques à ta guise dans les champs. Clique sur Écouter le dialogue : les voix vont alterner pour lire le texte. Astuces Tu peux ajouter/retirer des répliques dans le code HTML. Les voix proposées dépendent de ton système et navigateur : pour plus de diversité, installe d’autres voix vocales françaises sur ton ordinateur. Ce système fonctionne dans la plupart des navigateurs modernes, directement, sans bibliothèque externe. Si tu veux gérer un dialogue plus long, ou ajouter la sélection des langues/effets, ça se modifie facilement : il suffit d’ajouter des textarea et d’ajuster le script. //
